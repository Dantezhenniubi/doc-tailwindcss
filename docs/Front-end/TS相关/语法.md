# 语法
本文介绍一些遇到的TS语法问题

## 类型标注
需要在函数后面进行返回值的类型标注：
::: CTcode
```ts{1}
export function preloadAll(): void {
  lightBgList.forEach(({ url }) => {
    new Image().src = url;
  });
  darkBgList.forEach(({ url }) => {
    new Image().src = url;
  });
}
```
:::
