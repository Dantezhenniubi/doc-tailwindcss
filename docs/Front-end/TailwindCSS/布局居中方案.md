## CSS 居中的 4 种方案（Tailwind 版）

### Flexbox（推荐，零副作用）

```vue:demo
<template>
  <div class="flex items-center justify-center h-[300px] w-[600px] border border-red-500 bg-gray-200 dark:bg-gray-800">
    <div v-for="c in ['bg-red-500','bg-green-500','bg-blue-500']" :key="c"
         class="w-20 h-20 rounded-lg shadow-md hover:shadow-xl transition-shadow">
    </div>
  </div>
</template>
```

**口诀：`flex` `items-center` `justify-center`**

### Grid（一维/二维都爽）

```vue:demo
<template>
  <div class="grid place-items-center h-[300px] w-[600px] border border-red-500 bg-gray-200 dark:bg-gray-800">
    <div v-for="c in ['bg-red-500','bg-green-500','bg-blue-500']" :key="c"
         class="w-20 h-20 rounded-lg shadow-md hover:shadow-xl transition-shadow">
    </div>
  </div>
</template>
```

**口诀：`grid` `place-items-center`（= `align-items center` + `justify-items center`）**

### 绝对定位 + Transform（无父级依赖）

```vue:demo
<template>
  <div class="relative h-[300px] w-[600px] border border-red-500 bg-gray-200 dark:bg-gray-800">
    <div v-for="(c,i) in ['bg-red-500','bg-green-500','bg-blue-500']" :key="c"
         :class="c"
         class="absolute top-1/2 left-1/2 -translate-x-1/2 -translate-y-1/2
                w-20 h-20 rounded-lg shadow-md hover:shadow-xl transition-shadow">
    </div>
  </div>
</template>
```

**口诀：`absolute` `top-1/2` `left-1/2` `-translate-x-1/2` `-translate-y-1/2`**

### 绝对定位 + margin（需固定尺寸）

```vue:demo
<template>
  <div class="relative h-[300px] w-[600px] border border-red-500 bg-gray-200 dark:bg-gray-800">
    <div v-for="(c,i) in ['bg-red-500','bg-green-500','bg-blue-500']" :key="c"
         :class="c"
         class="absolute inset-0 m-auto w-20 h-20 rounded-lg shadow-md hover:shadow-xl transition-shadow">
    </div>
  </div>
</template>
```

**口诀：`absolute` `inset-0` `m-auto` `w-XX` `h-XX`**

## 绝对定位的缺点

**一句话：脱离文档流 → 布局脆弱、无法响应、堆叠上下文污染。**

| 场景             | 缺点                         | 推荐替代                        |
| ---------------- | ---------------------------- | ------------------------------- |
| **响应式卡片**   | 尺寸变化 → 居中失效          | `flex` / `grid`                 |
| **弹窗/Popover** | 需要跟随视口                 | `transform` + `position: fixed` |
| **大量元素**     | 堆叠上下文污染，z-index 地狱 | `grid` 或 `flex`                |

**结论：只在「弹窗、固定角标、必须脱离流」时才用绝对定位；日常居中优先 flex/grid。**

## 常用属性说明

> `align-items`、`justify-items`、`justify-content`、`align-self`、`justify-self` 都是什么？

::: success 一句话记忆
**「items」→ 单格内对齐；「content」→ 整行/整列对齐；「self」→ 单元素覆盖。**
:::

| 属性              | 作用轴          | 功能                            | Tailwind 等价                                             |
| ----------------- | --------------- | ------------------------------- | --------------------------------------------------------- |
| `align-items`     | **交叉轴**（Y） | **单行/单列** 内元素垂直对齐    | `items-start` / `items-center` / `items-end`              |
| `justify-items`   | **主轴**（X）   | **Grid 单格** 内元素水平对齐    | `justify-items-start` / `center` / `end`                  |
| `justify-content` | **主轴**（X）   | **整行/整列** 在容器内分布      | `justify-start` / `center` / `between` / `around` / `end` |
| `align-content`   | **交叉轴**（Y） | **整行/整列** 在容器内分布      | `content-start` / `center` / `between` / `around` / `end` |
| `align-self`      | **交叉轴**（Y） | **单元素** 覆盖 `align-items`   | `self-start` / `self-center` / `self-end`                 |
| `justify-self`    | **主轴**（X）   | **单元素** 覆盖 `justify-items` | `justify-self-start` / `center` / `end`                   |

**口诀：`items` 对单格，`content` 对整行，`self` 对单元素。**

## Tailwind有什么比较流行的解决方案？

| 场景           | 流行类名                              | 说明               |
| -------------- | ------------------------------------- | ------------------ |
| **一行居中**   | `flex items-center justify-center`    | 万能居中           |
| **Grid 居中**  | `grid place-items-center`             | 一维/二维都爽      |
| **绝对居中**   | `absolute inset-0 m-auto w-XX h-XX`   | 需固定尺寸         |
| **响应式断点** | `md:flex-row lg:grid-cols-3`          | 断点即类名         |
| **暗黑模式**   | `dark:bg-gray-800 dark:text-gray-100` | 自动跟随系统       |
| **动画/过渡**  | `transition-shadow duration-300`      | 只过渡真正变的属性 |

::: success
「Flex 日常，Grid 二维，绝对定位只救急；
items 对单格，content 对整行，self 对单元素；
:::

## Flex和Grid中区别

**着重谈一下`justify-items`和`justify-content`**

- 「justify-items ≈ 在 Grid 里用，Flex 里几乎无效；
- justify-content ≈ 在 Flex 和 Grid 里都能用，但 Flex 里叫『整行分布』，Grid 里叫『整列分布』。」

| 属性              | Flex 子项？ | Flex 整行？ | Grid 子项？ | Grid 整列？ | 一句话                  |
| ----------------- | ----------- | ----------- | ----------- | ----------- | ----------------------- |
| `justify-items`   | ❌ 无效      | ❌ 无效      | ✅ 有效      | ❌ 无效      | 「Grid 格子内子项对齐」 |
| `justify-content` | ✅ 有效      | ✅ 有效      | ✅ 有效      | ✅ 有效      | 「整行/整列分布」       |

---

> ✅ Flex 场景（一眼表）

| 属性                             | Tailwind 类            | Flex 结果                    |
| -------------------------------- | ---------------------- | ---------------------------- |
| `justify-content: center`        | `justify-center`       | **整行**在容器内居中         |
| `justify-content: space-between` | `justify-between`      | **整行**在容器内两端分布     |
| `align-items: center`            | `items-center`         | **所有子项**在整行内垂直居中 |
| `justify-items: center`          | `justify-items-center` | **无效**（Flex 无格子）      |

::: success 口诀
Flex 口诀：「justify-content 对整行，align-items 对所有子项，justify-items 无效」
:::

---

> ✅ Grid 场景（一眼表）

| 属性                             | Tailwind 类            | Grid 结果                    |
| -------------------------------- | ---------------------- | ---------------------------- |
| `justify-content: center`        | `justify-center`       | **整列**在容器内居中         |
| `justify-content: space-between` | `justify-between`      | **整列**在容器内两端分布     |
| `justify-items: center`          | `justify-items-center` | **所有子项**在格子里居中     |
| `align-items: center`            | `items-center`         | **所有子项**在格子里垂直居中 |

::: success 口诀
Grid 口诀：「justify-content 对整列，justify-items 对格子，align-items 对格子」
:::


::: danger 总结
**「Flex：justify-content 对整行，align-items 对所有子项，justify-items 无效；**

**Grid：justify-content 对整列，justify-items 对格子，align-items 对格子。」**
:::
